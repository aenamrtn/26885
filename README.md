# Setup Documentation

## What Tools I Installed
- Cursor IDE
- Claude Code extension (by Anthropic)
- Codex extension (by OpenAI)
- GitHub

## What Steps I Completed
- Downloaded and installed the Cursor application on my Mac.
- Searched the marketplace to install the Anthropic Claude Code extension.
- Installed the OpenAI Codex extension and authenticated my account.
- Created a public GitHub repository, initialized a README, and cloned it into Cursor.

## What Issues I Ran Into and How I Solved Them

**Issue 1:** I initially tried looking for Cursor extensions in my regular web browser dashboard.
**Solution:** I realized I needed to open the Extensions panel (`Cmd + Shift + X`) inside the actual Cursor Desktop application.

**Issue 2:** I couldn't find the sign-in button for Codex after installing it.
**Solution:** I learned how to use the Command Palette (`Cmd + Shift + P`) to open the specific Codex chat view, which revealed the hidden sign-in portal.
